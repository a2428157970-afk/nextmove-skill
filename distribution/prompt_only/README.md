# NextMove Prompt-only Pilot Kit

**NEXTMOVE_PROMPT_ONLY**

This kit contains prompts, instructions, output-reading guidance, and fictional
samples for an experience preview. It does not include the Python `skill/`
implementation and will not execute local Python Skill code.

Uploading these files to ChatGPT or another Agent does not install NextMove,
does not call `NextMoveSkill.run()`, and must never be described as Runtime
execution. Real NextMove execution requires the Runtime Skill Package and a
host environment with local Python 3.11+ execution.

Start with [QUICK_START.md](QUICK_START.md), then use [PROMPTS.md](PROMPTS.md).
Read [OUTPUT_GUIDE.md](OUTPUT_GUIDE.md) and
[PRIVACY_AND_FEEDBACK.md](PRIVACY_AND_FEEDBACK.md) before sharing feedback.
