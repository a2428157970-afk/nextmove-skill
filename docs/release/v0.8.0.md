# NextMove Skill v0.8.0

NextMove v0.8.0 is Developer Preview Ready. It is the operational-readiness release for Agent discovery, installation, and offline invocation. Creating the Git tag and public release remains a separate human-approved action.

## Release capabilities

- Discover five public capabilities through `skill.json` and Agent Tool metadata.
- Install locally with `python -m pip install -e .`.
- Invoke `career_analysis` through Python, Agent Runtime, `python -m skill`, or the `nextmove` command.
- Receive structured, JSON-serializable success and failure responses.
- Run without a network request, credential lookup, Web service, or Application Layer dependency.

## Entrypoints

- Agent and Python consumers invoke `NextMoveSkill.run`.
- CLI consumers invoke `skill.__main__:main` through `python -m skill` or the installed `nextmove` command.
- The legacy manifest `entrypoint` remains the CLI entrypoint for compatibility; `skill_entrypoint` and `cli_entrypoint` make both roles explicit.

## CLI

```bash
nextmove analyze --resume resume.txt --job-description "Backend role requiring Python and SQL."
```

The CLI requires a UTF-8 text resume. The job description is optional only at the CLI boundary and is normalized to an empty string when omitted. Python and Agent calls retain the required `job_description` field.

## Validation

NextMove requires Python 3.11 or later, pip, and setuptools 61 or later. Before publishing, run:

```bash
python -m pip install -e .
nextmove --help
python examples/skill_demo.py
python -m unittest discover -s tests
python -m compileall skill tests examples
```

On an offline machine, a compatible setuptools build backend must already be installed; pip build isolation may otherwise attempt package-index resolution. NextMove has no runtime package dependencies.

Repository-level compatibility is recorded in [Agent Compatibility](../agent-compatibility.md). ChatGPT Agent, Claude, and Cursor external invocation remains pending and is not a release-readiness claim.

## Known limitations

- Career intelligence is deterministic decision support and requires human review.
- The CLI does not extract PDF or Word files.
- No hosted endpoint, database, authentication system, or frontend is included.
- `--job-description-file` is not available in v0.8.0; it remains a post-release UX candidate.
