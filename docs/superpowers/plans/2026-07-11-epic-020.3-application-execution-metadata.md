# Epic-020.3 Application Execution Metadata Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a safe, immutable, transport-neutral execution-metadata contract to successful and failed valid career-analysis workflow responses.

**Architecture:** `ExecutionMetadata` is an Application-owned frozen schema. `CareerAnalysisService` creates a started instance only after request validation and uses immutable replacement to attach a completed or failed terminal instance to the existing workflow response. The workflow and `NextMoveSkill.run()` receive no metadata and keep their fixed four-step behavior.

**Tech Stack:** Python standard library (`dataclasses`, `datetime`, `uuid`, `json`, `unittest`); existing application schemas, service, workflow, and public Skill contracts; no new dependencies.

## Global Constraints

- Add no HTTP, FastAPI, frontend, database, provider, logging SDK, telemetry SDK, or dependency.
- Keep `application -> public skill API -> Skill Core`; `skill/` must never import `application/`.
- Keep execution metadata out of Skill Core, AI providers, backend, frontend, prompts, credentials, resume content, job-description content, raw inputs, and provider responses.
- Use timezone-aware UTC `datetime` values only. Obtain current times with `datetime.now(timezone.utc)`; never use naive `datetime.now()`.
- Serialize times with `datetime.isoformat()` to ISO 8601 strings; do not serialize numeric timestamps or use a third-party date library.
- Preserve request validation behavior, optional-job-description normalization, fixed capability order, fail-fast behavior, structured `SkillError`, and older `ApplicationResponse` serialization when metadata is absent.
- Do not modify the v0.7.0 tag or release-history entry. Do not push or merge during implementation.

# application/schemas/execution.py
from dataclasses import dataclass
from datetime import datetime, timezone


_ALLOWED_STATUSES = {"started", "completed", "failed"}


@dataclass(frozen=True, slots=True)
class ExecutionMetadata:
    execution_id: str
    workflow_name: str
    status: str
    started_at: datetime
    completed_at: datetime | None = None
    failed_step: str | None = None

    def __post_init__(self) -> None:
        if not self.execution_id:
            raise ValueError("execution_id must not be empty")
        if not self.workflow_name:
            raise ValueError("workflow_name must not be empty")
        if self.status not in _ALLOWED_STATUSES:
            raise ValueError("status must be started, completed, or failed")
        self._validate_utc("started_at", self.started_at)
        if self.completed_at is not None:
            self._validate_utc("completed_at", self.completed_at)
            if self.completed_at < self.started_at:
                raise ValueError("completed_at must not be before started_at")
        if self.status == "started" and (self.completed_at is not None or self.failed_step is not None):
            raise ValueError("started metadata must not have completed_at or failed_step")
        if self.status == "completed" and (self.completed_at is None or self.failed_step is not None):
            raise ValueError("completed metadata requires completed_at and no failed_step")
        if self.status == "failed" and (self.completed_at is None or not self.failed_step):
            raise ValueError("failed metadata requires completed_at and failed_step")

    @staticmethod
    def _validate_utc(field_name: str, value: datetime) -> None:
        if value.tzinfo is None or value.utcoffset() != timezone.utc.utcoffset(value):
            raise ValueError(f"{field_name} must be timezone-aware UTC")

    def to_dict(self) -> dict[str, object]:
        return {
            "execution_id": self.execution_id,
            "workflow_name": self.workflow_name,
            "status": self.status,
            "started_at": self.started_at.isoformat(),
            "completed_at": None if self.completed_at is None else self.completed_at.isoformat(),
            "failed_step": self.failed_step,
        }
```

# application/schemas/career.py
from application.schemas.execution import ExecutionMetadata

@dataclass(slots=True)
class ApplicationResponse:
    success: bool
    result: CareerAnalysisReport | None = None
    error_code: str | None = None
    failed_step: str | None = None
    message: str | None = None
    error: SkillError | None = None
    metadata: ExecutionMetadata | None = None

    def to_dict(self) -> dict[str, object]:
        if self.success:
            serialized = {"success": True, "result": to_dict(self.result)}
        else:
            serialized = {
                "success": False,
                "error_code": self.error_code,
                "failed_step": self.failed_step,
                "message": self.message,
                "error": to_dict(self.error),
            }
        if self.metadata is not None:
            serialized["metadata"] = self.metadata.to_dict()
        return serialized

# application/services/career_analysis.py
from dataclasses import replace
from datetime import datetime, timezone
from uuid import uuid4

from application.schemas import ApplicationResponse, CareerAnalysisRequest, ExecutionMetadata

class CareerAnalysisService:
    def analyze(self, request: CareerAnalysisRequest) -> ApplicationResponse:
        try:
            request.validate()
        except ValueError as error:
            message = str(error)
            return ApplicationResponse(
                success=False,
                error_code="APPLICATION_VALIDATION_ERROR",
                failed_step="request_validation",
                message=message,
                error=SkillError(
                    code="APPLICATION_VALIDATION_ERROR",
                    message=message,
                ),
            )

        started_metadata = ExecutionMetadata(
            execution_id=uuid4().hex,
            workflow_name="career_analysis",
            status="started",
            started_at=datetime.now(timezone.utc),
        )
        response = self.workflow.run(request.resume, request.normalized_job_description())
        completed_at = datetime.now(timezone.utc)
        metadata = replace(
            started_metadata,
            status="completed" if response.success else "failed",
            completed_at=completed_at,
            failed_step=None if response.success else response.failed_step,
        )
        return replace(response, metadata=metadata)
```
