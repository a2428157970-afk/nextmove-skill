# Epic-025.4 Career Intelligence Quality Benchmark Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an offline, synthetic Career Intelligence Quality Benchmark that measures matching, evidence, explanation, and safety quality without changing production behavior or public contracts.

**Architecture:** Keep the benchmark entirely in test, fixture, script, and quality-documentation paths. A test-only adapter constructs `ResumeProfile`, calls the existing domain classifier, career-stage assessor, internal matcher explanation path, and public matcher facade, then evaluates atomic fixture expectations. Automated semantic metrics produce a content-safe report; safety checks enter regression immediately, while unsupported semantic expectations remain visible benchmark failures for a separately approved remediation Epic.

**Tech Stack:** Python standard library, dataclasses, JSON, `unittest`, existing `skill` internal APIs; no new dependency, provider, transport, or network call.

## Global Constraints

- Do not modify `NextMoveSkill.run()`, `skill/interface.py`, `JobMatchResult`, `skill/agent/`, the Application Layer, CLI, providers, dependency manifests, release files, or Git tags.
- Do not modify any file under `skill/`; the benchmark observes the current core and reports gaps.
- Preserve exactly these public fields and serialized keys: `match_score`, `matched_skills`, `missing_skills`, `strengths`, `gaps`, `recommendations`.
- Use only fictional structured profiles. Fixtures contain no personal information, raw resume text, real employer, link, endpoint, token, or credential.
- Keep benchmark observations transient. Serialized reports contain only case IDs, metric scores, failed check names, thresholds, and aggregate counts.
- Treat `unknown != missing`, no invented experience, and no unsupported claims as non-compensating hard safety gates.
- Use concept/category assertions, not exact explanation prose snapshots.
- Use TDD: test red, minimal implementation, test green, and focused commits.
- A failing semantic benchmark is a valid finding. Do not weaken expected quality or modify production code within this Epic to force a green report.

from __future__ import annotations

from dataclasses import asdict, dataclass
import json
from pathlib import Path
from typing import Any


@dataclass(frozen=True, slots=True)
class ExpectedRequirement:
    name: str
    status: str
    evidence_concepts: tuple[str, ...] = ()


@dataclass(frozen=True, slots=True)
class ScenarioExpected:
    resume_domain: str
    target_domain: str
    resume_family: str | None
    target_family: str | None
    career_stage: str
    requirements: tuple[ExpectedRequirement, ...]
    strength_concepts: tuple[str, ...]
    gap_concepts: tuple[str, ...]
    risk_categories: tuple[str, ...]
    forbidden_conclusions: tuple[str, ...]


@dataclass(frozen=True, slots=True)
class CareerIntelligenceCase:
    case_id: str
    description: str
    profile: dict[str, Any]
    target_role: str
    job_description: str
    expected: ScenarioExpected


def load_cases(directory: Path) -> list[CareerIntelligenceCase]:
    cases: list[CareerIntelligenceCase] = []
    for path in sorted(directory.glob("*.json")):
        payload = json.loads(path.read_text(encoding="utf-8"))
        expected = payload["expected"]
        requirements = tuple(
            ExpectedRequirement(
                name=item["name"],
                status=item["status"],
                evidence_concepts=tuple(item.get("evidence_concepts", [])),
            )
            for item in expected["requirements"]
        )
        cases.append(
            CareerIntelligenceCase(
                case_id=payload["case_id"],
                description=payload["description"],
                profile=payload["profile"],
                target_role=payload["target"]["role"],
                job_description=payload["target"]["job_description"],
                expected=ScenarioExpected(
                    resume_domain=expected["resume_domain"],
                    target_domain=expected["target_domain"],
                    resume_family=expected.get("resume_family"),
                    target_family=expected.get("target_family"),
                    career_stage=expected["career_stage"],
                    requirements=requirements,
                    strength_concepts=tuple(expected["strength_concepts"]),
                    gap_concepts=tuple(expected["gap_concepts"]),
                    risk_categories=tuple(expected["risk_categories"]),
                    forbidden_conclusions=tuple(expected["forbidden_conclusions"]),
                ),
            )
        )
    return cases

~~~python
from skill.career.stage_assessor import CareerStageAssessor
from skill.career.stages import CareerStageAssessment
from skill.matching.classifier import DomainClassifier
from skill.matching.explanations import MatchExplanationResult
from skill.matching.matcher import JobMatcher
from skill.matching.schemas import DomainClassification, JobMatchResult, MatchAssessment
from skill.schemas.resume import EducationEntry, ExperienceEntry, ProjectEntry, ResumeProfile


@dataclass(frozen=True, slots=True)
class CareerIntelligenceObservation:
    resume_domain: DomainClassification
    target_domain: DomainClassification
    stage: CareerStageAssessment
    assessment: MatchAssessment
    explanation: MatchExplanationResult
    public_result: JobMatchResult
    professional_evidence: tuple[str, ...]


def _profile(payload: dict[str, Any]) -> ResumeProfile:
    return ResumeProfile(
        summary=payload.get("summary"),
        skills=list(payload.get("skills", [])),
        experience=[ExperienceEntry(**item) for item in payload.get("experience", [])],
        education=[EducationEntry(**item) for item in payload.get("education", [])],
        projects=[ProjectEntry(**item) for item in payload.get("projects", [])],
        certifications=list(payload.get("certifications", [])),
        languages=list(payload.get("languages", [])),
    )


def _professional_evidence(profile: ResumeProfile) -> tuple[str, ...]:
    values: list[str] = [profile.summary or "", *profile.skills]
    for entry in profile.experience:
        values.extend((entry.role or "", *entry.highlights))
    for entry in profile.education:
        values.extend(value or "" for value in (entry.degree, entry.field, entry.description))
    for entry in profile.projects:
        values.extend((entry.name or "", entry.description or "", *entry.technologies))
    values.extend(profile.certifications)
    values.extend(profile.languages)
    return tuple(value for value in values if value)


def observe_case(case: CareerIntelligenceCase) -> CareerIntelligenceObservation:
    profile = _profile(case.profile)
    classifier = DomainClassifier()
    matcher = JobMatcher()
    assessment, explanation = matcher._assess_and_explain(profile, case.job_description)
    return CareerIntelligenceObservation(
        resume_domain=classifier.classify_profile(profile),
        target_domain=classifier.classify_text(case.job_description),
        stage=CareerStageAssessor().assess(profile),
        assessment=assessment,
        explanation=explanation,
        public_result=matcher.match(profile, case.job_description),
        professional_evidence=_professional_evidence(profile),
    )
~~~

~~~python
@dataclass(frozen=True, slots=True)
class BenchmarkMetric:
    name: str
    score: int | None


@dataclass(frozen=True, slots=True)
class SafetyGate:
    name: str
    passed: bool


@dataclass(frozen=True, slots=True)
class CareerIntelligenceEvaluation:
    case_id: str
    success: bool
    score: int
    metrics: tuple[BenchmarkMetric, ...]
    safety_gates: tuple[SafetyGate, ...]
    failures: tuple[str, ...]

    def to_dict(self) -> dict[str, Any]:
        return asdict(self)


def _ratio(checks: list[bool]) -> int | None:
    if not checks:
        return None
    return round(100 * sum(checks) / len(checks))


def _contains_concept(values: tuple[str, ...], concept: str) -> bool:
    needle = concept.casefold()
    return any(needle in value.casefold() for value in values)


def _explanation_texts(observation: CareerIntelligenceObservation) -> tuple[str, ...]:
    items = (
        *observation.explanation.strengths,
        *observation.explanation.gaps,
        *observation.explanation.risks,
    )
    return tuple(
        value
        for item in items
        for value in (item.summary, *item.related_requirements)
    )


def evaluate_case(
    case: CareerIntelligenceCase,
    observation: CareerIntelligenceObservation,
) -> CareerIntelligenceEvaluation:
    expected = case.expected
    actual_requirements = {
        item.requirement: item for item in observation.explanation.requirements
    }
    domain_checks = [
        observation.resume_domain.domain.value == expected.resume_domain,
        observation.target_domain.domain.value == expected.target_domain,
        observation.stage.stage.value == expected.career_stage,
    ]
    if expected.resume_family is not None:
        domain_checks.append(
            observation.resume_domain.job_family is not None
            and observation.resume_domain.job_family.value == expected.resume_family
        )
    if expected.target_family is not None:
        domain_checks.append(
            observation.target_domain.job_family is not None
            and observation.target_domain.job_family.value == expected.target_family
        )
    requirement_checks = [
        requirement.name in actual_requirements
        and actual_requirements[requirement.name].status.value == requirement.status
        for requirement in expected.requirements
    ]
    relevance_checks: list[bool] = []
    for requirement in expected.requirements:
        if not requirement.evidence_concepts:
            continue
        actual = actual_requirements.get(requirement.name)
        relevance_checks.append(
            actual is not None
            and all(
                _contains_concept(tuple(item.text for item in actual.evidence), concept)
                for concept in requirement.evidence_concepts
            )
            and all(
                item.source
                in {"summary", "experience", "project", "skill", "education", "certification", "language"}
                for item in actual.evidence
            )
        )
    retained_evidence = tuple(
        item.text
        for requirement in observation.explanation.requirements
        for item in requirement.evidence
    )
    grounding_checks = [
        evidence in observation.professional_evidence for evidence in retained_evidence
    ]
    strength_values = tuple(
        value
        for item in observation.explanation.strengths
        for value in (item.summary, *item.related_requirements)
    )
    gap_values = tuple(
        value
        for item in observation.explanation.gaps
        for value in (item.summary, *item.related_requirements)
    ) + tuple(observation.public_result.gaps)
    risk_values = tuple(item.category for item in observation.explanation.risks)
    strength_checks = [
        _contains_concept(strength_values, concept)
        for concept in expected.strength_concepts
    ]
    gap_checks = [
        _contains_concept(gap_values, concept)
        for concept in expected.gap_concepts
    ]
    risk_checks = [category in risk_values for category in expected.risk_categories]

    unknown_names = {
        item.requirement
        for item in observation.explanation.requirements
        if item.status.value == "unknown"
    } | {
        item.name for item in expected.requirements if item.status == "unknown"
    }
    unknown_not_missing = not unknown_names.intersection(
        observation.public_result.missing_skills
    ) and not any(
        item.category == "missing_capability"
        and unknown_names.intersection(item.related_requirements)
        for item in observation.explanation.gaps
    )
    no_invented_experience = all(grounding_checks) if grounding_checks else True
    searchable = " ".join(
        (
            *_explanation_texts(observation),
            *observation.public_result.matched_skills,
            *observation.public_result.missing_skills,
            *observation.public_result.strengths,
            *observation.public_result.gaps,
            *observation.public_result.recommendations,
        )
    ).casefold()
    no_unsupported_claims = not any(
        phrase.casefold() in searchable
        for phrase in expected.forbidden_conclusions
    )

    metrics = (
        BenchmarkMetric("domain_accuracy", _ratio(domain_checks)),
        BenchmarkMetric("requirement_coverage", _ratio(requirement_checks)),
        BenchmarkMetric("evidence_relevance", _ratio(relevance_checks)),
        BenchmarkMetric("evidence_grounding", _ratio(grounding_checks)),
        BenchmarkMetric("strength_correctness", _ratio(strength_checks)),
        BenchmarkMetric("gap_correctness", _ratio(gap_checks)),
        BenchmarkMetric("risk_correctness", _ratio(risk_checks)),
    )
    safety_gates = (
        SafetyGate("unknown_not_missing", unknown_not_missing),
        SafetyGate("no_invented_experience", no_invented_experience),
        SafetyGate("no_unsupported_claims", no_unsupported_claims),
    )
    applicable_scores = [metric.score for metric in metrics if metric.score is not None]
    score = round(sum(applicable_scores) / len(applicable_scores)) if applicable_scores else 0
    failures = tuple(
        [metric.name for metric in metrics if metric.score is not None and metric.score < 80]
        + [gate.name for gate in safety_gates if not gate.passed]
        + (["scenario_average"] if score < 85 else [])
    )
    return CareerIntelligenceEvaluation(
        case_id=case.case_id,
        success=not failures,
        score=score,
        metrics=metrics,
        safety_gates=safety_gates,
        failures=failures,
    )
~~~

from __future__ import annotations

import argparse
import json
from pathlib import Path
import sys

ROOT = Path(__file__).resolve().parents[1]
if str(ROOT) not in sys.path:
    sys.path.insert(0, str(ROOT))

from tests.quality.career_intelligence_benchmark import evaluate_case, load_cases, observe_case


def build_report() -> dict[str, object]:
    directory = ROOT / "tests" / "fixtures" / "career_intelligence"
    results = [evaluate_case(case, observe_case(case)) for case in load_cases(directory)]
    return {
        "summary": {
            "total_cases": len(results),
            "passed_cases": sum(result.success for result in results),
            "failed_cases": sum(not result.success for result in results),
            "network_calls": 0,
        },
        "cases": [result.to_dict() for result in results],
    }


def main(argv: list[str] | None = None) -> int:
    parser = argparse.ArgumentParser(description="Run the offline Career Intelligence benchmark.")
    parser.add_argument("--format", choices=("json", "text"), default="text")
    args = parser.parse_args(argv)
    report = build_report()
    if args.format == "json":
        print(json.dumps(report, ensure_ascii=False, indent=2))
    else:
        summary = report["summary"]
        print(
            "Career Intelligence Benchmark: "
            f"{summary['passed_cases']}/{summary['total_cases']} passed; "
            f"{summary['failed_cases']} failed."
        )
        for case in report["cases"]:
            print(f"- {case['case_id']}: score={case['score']} failures={','.join(case['failures']) or 'none'}")
    return 0 if report["summary"]["failed_cases"] == 0 else 1


if __name__ == "__main__":
    raise SystemExit(main())

~~~markdown
# Career Intelligence Review Checklist

Use only synthetic benchmark cases. Do not record resume or job-description text, evidence snippets, explanation prose, identity, employer, contact information, credentials, or free-form private data.

## Scoring anchors

- **1 — Unacceptable:** materially incorrect, invented, unsafe, or unusable.
- **3 — Adequate:** main conclusion is correct but loses relevant nuance.
- **5 — Excellent:** precise, evidence-grounded, calibrated, complete, and actionable.

## Criteria

1. **Candidate background:** domain, career stage, transferable context, and uncertainty are understood correctly.
2. **Evidence grounding:** every material conclusion can be traced to supplied professional evidence.
3. **Match calibration:** direct, partial, missing, and unknown evidence are distinguished without exaggeration.
4. **Risk completeness:** material evidence and transition risks are identified without unsupported risk claims.
5. **Actionability:** recommended actions are truthful, feasible, specific, and connected to a gap or risk.

## Acceptance

The average score must be at least 4.0, no criterion may score below 3, and every automated safety gate must pass.

## Content-safe review record

Record only: `case_id`, five integer scores, `reviewer_role`, `review_date`, optional reason codes from `background`, `grounding`, `calibration`, `risk`, and `action`, plus final `passed`. Do not add free-form source or output content.
~~~
