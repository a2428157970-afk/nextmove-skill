# Human Career Report Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an internal, evidence-grounded `HumanCareerReport` that presents existing NextMove career intelligence in a five-section user journey without changing any public output.

**Architecture:** Add an internal `skill/reporting/` package containing immutable report schemas, bounded Simplified Chinese language policy, and a deterministic aggregation builder. The builder consumes five completed assessments and maps them into a JSON-serializable report; it does not invoke or duplicate matching, career-stage, explanation, or transition logic.

**Tech Stack:** Python 3.11+, standard-library dataclasses/enums, existing `skill.utils.to_dict`, `unittest`, existing offline Career Intelligence benchmark.

## Global Constraints

- Do not modify `JobMatchResult`, `CareerAnalysisReport`, `NextMoveSkill.run()`, Agent Contract, Application Layer, CLI, providers, frontend, or backend.
- Do not add databases, network calls, paid APIs, or new dependencies.
- Do not reimplement matching, career-stage, transferable-skill, explanation, transition, risk, or action decisions.
- Keep `HumanCareerReport` internal: no export from `skill/__init__.py`, `skill/schemas/`, or `skill/agent/`, and no insertion into an existing response.
- Use deterministic Simplified Chinese presentation copy with English field and enum identifiers.
- Every displayed strength must contain at least one safe source evidence item.
- Preserve `unknown`, `partial`, `missing`, direct, and transferable meanings; never turn missing evidence into a proven inability.
- Keep all existing public serialized key sets unchanged.
- Complete Epic closeout records only after all implementation and regressions are green.

Create `skill/reporting/language.py` with one normalized phrase scan and fixed

```python
class UnsafeReportLanguage(ValueError):
    pass


class HumanReportLanguage:
    _FORBIDDEN = (
        "你一定成功", "你不适合", "你无法转型", "保证拿 offer",
        "guaranteed offer", "unsuitable", "cannot transition",
    )

    def is_safe(self, text: str) -> bool:
        normalized = " ".join(text.casefold().split())
        return not any(phrase.casefold() in normalized for phrase in self._FORBIDDEN)

    def ensure_safe_generated(self, text: str) -> str:
        if not self.is_safe(text):
            raise UnsafeReportLanguage("generated Human Career Report copy is unsafe")
        return text

    def career_summary(self, domain: str, stage: str) -> str:
        return self.ensure_safe_generated(
            f"当前证据显示，你的职业积累主要在 {domain}，职业阶段为 {stage}。"
        )

    def partial_gap(self, capability: str) -> str:
        return self.ensure_safe_generated(
            f"{capability} 已有部分相关证据，建议验证直接负责范围。"
        )

    def unknown_gap(self, capability: str) -> str:
        return self.ensure_safe_generated(
            f"简历中尚未看到 {capability} 的直接证据，可以优先补充相关案例。"
        )
```

Add templates for stage, job fit, transition type, missing conflict,
development area, risk verification, low confidence, and empty action horizon.
Every template must call `ensure_safe_generated` before returning.

Create the facade with no assessor dependencies:

```python
class HumanCareerReportBuilder:
    def __init__(self, language: HumanReportLanguage | None = None) -> None:
        self.language = language or HumanReportLanguage()

    def build(
        self,
        resume_analysis: ResumeAnalysisResult,
        job_match: JobMatchResult,
        career_stage: CareerStageAssessment,
        match_explanation: MatchExplanationResult,
        career_transition: CareerTransitionAssessment,
    ) -> HumanCareerReport:
        strengths = self._strengths(match_explanation)
        gaps = self._gaps(match_explanation, career_transition)
        return HumanCareerReport(
            career_summary=self._career_summary(career_stage, career_transition, job_match, match_explanation),
            current_profile=self._current_profile(resume_analysis, career_transition, strengths),
            career_stage=self._career_stage(career_stage),
            strengths=strengths,
            job_fit=self._job_fit(job_match, match_explanation, gaps),
            capability_gaps=gaps,
            transition_path=self._transition_path(career_transition),
            action_plan=self._action_plan(career_transition),
            risks=self._risks(match_explanation, career_transition),
            confidence=self._confidence(career_stage, match_explanation, career_transition),
        )

from pathlib import Path

from benchmark.evaluator import _profile_from_fixture
from benchmark.loader import load_scenarios


SCENARIO_DIR = Path(__file__).parent / "benchmark" / "scenarios"


def build_report(
    scenario_id: str,
    *,
    target_role: str | None = None,
    target_jd: str | None = None,
) -> HumanCareerReport:
    scenario = next(
        item for item in load_scenarios(SCENARIO_DIR)
        if item.scenario_id == scenario_id
    )
    profile = _profile_from_fixture(scenario.resume_fixture)
    role = target_role or scenario.target_role
    job_description = target_jd or scenario.target_jd
    matcher = JobMatcher()
    assessment, explanation = matcher._assess_and_explain(profile, job_description)
    classifier = DomainClassifier()
    current = classifier.classify_profile(profile)
    target = classifier.classify_text(job_description)
    stage = CareerStageAssessor().assess(profile)
    transition = CareerTransitionAssessor().assess(
        current,
        target,
        stage,
        assessment.transferability,
        explanation,
        TargetRoleLevelAssessor().classify(f"{role} {job_description}"),
    )
    return HumanCareerReportBuilder().build(
        ResumeAnalyzer().analyze(profile),
        matcher.match(profile, job_description),
        stage,
        explanation,
        transition,
    )
```
