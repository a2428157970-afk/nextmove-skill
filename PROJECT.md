# NextMove AI Project Config

## Mission
Help users make better career decisions with AI, instead of helping them blindly submit more resumes.

## Product
NextMove is an AI Career Intelligence Platform and AI Career Intelligence Skill Framework.

Slogan: Know Your Value. Find Your Next Career Move.

## Current Stage
Epic-021 Skill Productization completed as NextMove Skill Developer Preview
v0.8.0. The installable Skill exposes `career_analysis`, machine-readable
metadata, Agent instructions, and an offline product demo.

## Current Sprint
Epic-021 Skill Productization completed.

## Current Goal
Validate the Skill Developer Preview with Agent consumers while keeping the
public capabilities stable and independent from Web and Application Layer
delivery concerns.

## Tech Stack
- Frontend: Next.js
- Backend: FastAPI
- Database: Not selected yet
- AI: Optional provider-neutral enhancement with an injected-transport
  OpenAI-compatible adapter; no SDK, environment loading, or live default call

## Out Of Scope
- No business features
- No login system
- No AI features
- No database
- No resume submission automation
